# AeroSim
I started AeroSim as a high school sophomore (2025-2026). I’m actively learning and welcome critiques, bug reports, and improvement ideas.
AeroSim is a student-built aerodynamic and rocket simulation platform started in 2025-2026 to make advanced flight analysis accessible, visual, and practical for students, hobbyists, and early-stage engineers. Feedback and critiques are welcome.
